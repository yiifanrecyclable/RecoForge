# RecoForge

RecoForge 是一款面向 3D 打印产品创作者的产品管理与成本核算工具。

它适合用来整理一个产品从拆件、打样、定价到版本迭代的全过程，帮助你把打印件、外协件、直采件、辅料、耗材和标件统一管理起来，减少对零散表格的依赖。

## 适合谁使用

- 有多个3D打印产品长期维护的创作者
- 3D打印农场
- 需要核算材料、电费、辅材、外协和采购成本的工作室。

## 你可以用它做什么

### 管理产品与版本

- 为不同产品建立独立档案。
- 按版本记录产品迭代、封面图、说明和状态。
- 在概览页快速浏览产品卡片、在售状态和资料统计。
- 在产品页查看总生产成本、打印工时和成本构成。

### 管理各种零件

- 分别管理打印件、定制件、直采件和辅料。
- 为零件设置编号、数量、备注和产品 / 包装属性。
- 复制已有零件快速新建相似项目。
- 在同一产品版本内避免重复编号。
- 批量选择打印件并统一变更主耗材。

### 计算产品成本

- 打印件可按重量、耗材价格、打印时间、电价、辅材和良率计算成本。
- 定制件、直采件和辅料可按单价与数量计算。
- 直采件可引用标件库价格，便于统一维护。
- 用圆环图查看产品成本由哪些部分构成。
- 使用盘计算器，把整盘打印数量、时间和耗材用量换算为单件数据。
- 可在产品页隐藏成本数字，适合演示或截图。

### 管理 STL 模型

- 为打印件关联 STL 模型。
- 自动生成模型缩略图。
- 在软件内预览 3D 模型，支持旋转、缩放和平移。
- 可用当前观察视角重新生成缩略图。

### 从 3MF 文件提取打印件

- 读取 3MF 中的模型对象名称。
- 将模型提取为可管理的打印件。
- 支持组合体作为独立零件导入。
- 通过模型内容识别重复 STL，避免重复保存。
- 名称重复但模型不同的零件，可选择替换或保留。

### 管理耗材和标件

- 建立自己的耗材库，记录品牌、系列、颜色和价格。
- 建立常用标件库，方便直采件引用。
- 支持图片上传、裁剪和替换，让资料更直观。
- 支持导入耗材预设包，快速建立常见品牌的耗材资料。
- 支持批量修改耗材单价。

### 管理工作区

- 切换到已有工作区继续使用。
- 将当前工作区迁移到新位置。
- 一键备份工作区，便于转移和留档。

## 本地数据

RecoForge 的数据和资源保存在本地工作区中。你可以根据自己的使用习惯设置工作区位置，方便备份、迁移和长期维护。
